# 🌉 NeonBridge
*Private chat + file sharing. Phone-to-phone. No towers. No tracking.*

✨ Your privacy is our policy! Keeping your name out your ISP's mouth, one file share at a time!

## What it does
- 🔐 E2E encrypted chats that vanish when closed (RAM-only)
- 📁 File sharing: save only what you choose → encrypted vault
- 🌡️ "Hot/cold" proximity meter (visual/haptic/audio)
- 🌉 Offline-first (Wi-Fi Direct), internet fallback *only if you consent*
- 🎮 16-bit serene UI • Press Start 2P font • soft chimes

## Threat model
✅ Protects against: ISP sniffing, app telemetry, metadata harvesting  
⚠️ Not for: state-level forensic labs or unlocked-device coercion  
💡 Goal: Make big-tech data collection *pointless*, not unbreakable

## Build it
1. Fork this repo
2. Open in Android Studio
3. Run on two Android devices (API 29+)
4. Scan QR to pair → feel the neon bridge 💙
